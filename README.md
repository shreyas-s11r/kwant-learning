KWANT Learning

Documenting my journey learning quantum tranport and simulations using the KWANT python package.

>Learning Goals

- Understand tight-binding Hamiltonians
- Learn the Kwant framework
- Simulate quantum transport in low-dimensional systems
- Develop intuition for Green's functions and the Landauer formalism
- Study graphene and other 2D materials
- Build simulations relevant to semiconductor and quantum devices

>Each notebook aims to answer four questions:

1. What problem are we solving?
2. What is the underlying physics?
3. How is it implemented in Kwant?
4. What do the simulation results mean?

>Progress

This repository will evolve as I learn. Early notebooks may be simple, but the long-term goal is to build realistic simulations of quantum and semiconductor devices.
